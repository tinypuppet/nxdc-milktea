### 简介

一套仿奈雪の茶小程序的项目模板。

如果你喜欢这套模板，请给个star呗~

### 说明

1. 本项目包含：

	- 首页
	- 点餐（自取和外卖两种方式，有基本的点餐逻辑处理）
	- 取餐
	- 我的
	- 积分商城
	- 积分商城详情页
	- 积分签到
	- 会员码
	- 我的卡券
	- 收货地址
	- 我的资料
	- 我的订单
	- 订单详情
	- 余额
	- 会员卷包
	- 卷包详情
	- 订单评价
	- 买茶送包
	- 更多服务
	- 开发票

2. 所有数据都是静态的，保存在api文件夹下。
